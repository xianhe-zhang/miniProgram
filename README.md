# miniProgram
基于前端三大件（HTML+CSS+JavaScript）的小程序，主要包括资讯浏览和电影推荐功能。

A mini-program run on Wechat based on HTML, CSS and JavaScript. Including trending news and movie exhibition.  

<br/>

__文件格式转化:__ 

    .wxss = .css
    
    .wxml = .html
    
    .js   = .js
    
<br/>
